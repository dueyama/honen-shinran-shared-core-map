Purpose
  The Japanese paper PDF is treated as the fixed public Japanese edition. The
  TeX file is recorded as the corresponding source file for that edition. The
  English PDF and HTML are generated from the English TeX, which is generated
  against the Japanese TeX structural source. At public release time, verify
  that the files stored at the release tag match the hashes above.

  Later corrections and validation notes should be published as separate
  supplements or errata, rather than silent rewrites of these fixed public
  artifacts.

  This is also part of the repository's experiment in publishing a humanities
  data-analysis paper through GitHub, GitHub Pages, GitHub Releases, and
  reproducible process records.
