公開境界

このページは公開用の短い出典・検証サマリです。詳細な制作・検証記録はリポジトリ内の memory.md と docs/source-provenance.md を参照してください。

本文・派生データ

  • raw/processed本文、chunk preview、embedding cache、embedding vectorは公開HTMLに含めない。
  • 図表、SAT行範囲、集計値、検証メモ、公開可能な派生情報のみを掲載する。
  • 本文提供元の利用条件は各提供元に従う。

  • SATなどから取得した元本文は本リポジトリでは再配布せず、本リポジトリのライセンス対象にも含めない。元本文の利用は各提供元の利用条件に従う。

主要入力

  • SAT大正新脩大藏經テキストデータベース。
  • 既存のUnicode-safe SAT漢文チャンクと既存embedding cache。
  • OpenAI text-embedding-3-large による3072次元埋め込み。

公開前検証

  • TeX PDFは uplatex と dvipdfmx で生成。
  • 日本語HTML論文は、PDFと同じ公開TeX原稿 docs/paper/honen-shinran-shared-core-paper.tex から生成する。
  • 英語TeXは、日本語TeXの構造を正本として scripts/build_english_tex_from_ja_tex.py で生成する。
  • 英語PDFと英語HTML論文は、同じ公開英語TeX原稿 docs/paper/honen-shinran-shared-core-paper-en.tex から生成する。
  • PDFはPyMuPDFでPNGレンダリングして目視確認。
  • 先行 Okyou repository は読み取り専用で参照し、変更しない。

リリース後の訂正

公開リリース後は、本文・PDF・主要HTMLをその時点の固定版として扱う。誤記、補足、解釈の変更、図表差し替えが必要な場合は、公開版を黙って置き換えず、Errataに記録する。

固定公開物

公開用PDFとTeXのSHA-256は checksums.txt に記録する。tag/release作成時には、このハッシュとrelease tag上のファイルが一致することを確認する。
